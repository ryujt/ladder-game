  # FilmTalk 요구사항 정의서

FilmTalk는 드라마, 애니메이션, 영화, 책 등 다양한 콘텐츠를 활용하여 사용자의 외국어 학습을 지원합니다. 사용자의 학습 이력을 기반으로 현재 수준을 자동으로 인식하고 맞춤형 학습 콘텐츠를 추천하는 기능을 포함한 확장된 외국어 학습 애플리케이션입니다.

## 주요 기능 목록

### 1. 콘텐츠 라이브러리 및 관리
- 다양한 미디어 제공: 드라마, 애니메이션, 영화, 책 등의 대본과 텍스트 제공
- 콘텐츠 분류 및 검색 기능: 장르, 제작 연도, 작가, 감독 등의 필터링을 통해 원하는 콘텐츠 검색 지원
- 사용자 제안 콘텐츠 관리: 커뮤니티를 통한 신규 콘텐츠 제안 및 업데이트 반영

### 2. 사전 학습 및 콘텐츠 분석
- 난이도별 단어 및 문장 자동 추출: 콘텐츠 내에서 난이도별 단어를 추출하여 플래시카드로 제공

### 3. 단어 및 문장 히스토리 관리와 복습 시스템
- 자동 저장 기능: 학습 중 틀린 단어나 문장을 자동으로 저장하여 복습 가능
- 간격 반복 알고리즘(SM2 알고리즘)을 이용한 효율적 복습 추천

### 4. 사용자 수준 자동 인식 및 추천
- 오답 히스토리 분석: 틀린 문장과 단어 기록을 분석하여 사용자의 언어 능력과 취약점 파악
- 단어 추천할 때 분석된 자료를 사용

### 5. 오프라인 모드 지원
- 콘텐츠 및 학습 자료 등을 로컬에 캐싱하여 재사용한다.

### 1. 로그인 및 회원가입
```navigation
Home --> Main : 로그인 성공
Home --> Login : 미로그인 상태
Login --> Terms : 신규 회원
Terms --> Signup : 약관 동의 완료
Signup --> Signup : 오류 발생
Signup --> Main
Login --> (signin) : 로그인 시도
(signin) --> Login : 오류 발생
(signin) --> Main
Main --> Content
Main --> ShuffleTest
Main --> Settings
```

### 2. 콘텐츠 선택 (드라마, 애니메이션, 영화, 책)
```navigation
Main --> (search) : 콘텐츠 검색
(search) --> Main : 검색 결과 표시
Main --> (selectCategory) : 콘텐츠 카테고리 선택
(selectCategory) --> Main : 선택한 카테고리 콘텐츠 표시
Main --> Content : 콘텐츠 상세 보기
```

### 3. 사전 학습 과정
```navigation
Main --> (search) : 콘텐츠 검색
(search) --> Main : 검색 결과 표시
Main --> (selectCategory) : 콘텐츠 카테고리 선택
(selectCategory) --> Main : 선택한 카테고리 콘텐츠 표시
Main --> Content : 콘텐츠 상세 보기
Content --> WordTest : 단어 학습 시작
Content --> SentenceTest : 문장 학습 시작
```

## 화면 구성(Screen layout)

* / --> 상하로 나뉨
* | --> 좌우로 나뉨

### Main
- 상단: 검색창 및 아이콘 버튼
- 콘텐츠 목록: 포스터 | (제목 / 설명)

### Settings
- 잔액 표시 및 충전
- 언어별 학습 레벨 설정
- 기타 설정

### Content
- 콘텐츠 정보: 포스터 | (제목 / 설명)
- 학습 이력 표시 영역 (스크롤 가능)
- 단어 학습 버튼
- 문장 학습 버튼

### WordTest
- 사용자의 학습 레벨에 따라 단어 목록 제공
- 최초 학습 후 틀린 단어만 별도 관리 및 표시
- 단어 표시 방식 (간격 반복 시스템 기반)
  - 왼쪽 스와이프 → 아는 단어 처리
  - 오른쪽 스와이프 → 모르는 단어 처리
  - 더블 클릭 → 단어 번역
- 오디오 재생 기능

### SentenceTest
- 사용자의 학습 레벨에 따라 문장 목록 제공
- 최초 학습 후 틀린 문장만 별도 관리 및 표시
- 문장 표시 방식 (간격 반복 시스템 기반)
  - 왼쪽 스와이프 → 아는 문장 처리
  - 오른쪽 스와이프 → 모르는 문장 처리
  - 더블 클릭 → 문장 번역
- 오디오 재생 기능

### ShuffleTest
- Content 구분없이 모든 단어장과 문장 목록으로 테스트 진행
- 반복 주기는 SM2 알고리즘 사용
